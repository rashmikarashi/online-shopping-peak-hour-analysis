# Online Shopping Peak Hour Analysis

## Project Description
This project analyzes online shopping order data to identify peak shopping hours
using Exploratory Data Analysis (EDA).

## Objectives
- To generate synthetic order data
- To analyze order frequency by hour
- To find peak shopping hours
- To visualize the data using line plots

## Tools Used
- Python
- Pandas
- NumPy
- Matplotlib
- VS Code

## How to Run the Project
1. Install required libraries:
   python -m pip install pandas numpy matplotlib
2. Run the script:
   python analysis.py

## Output
The output includes descriptive statistics and a line graph showing
the busiest shopping hours.

# Online Shopping Peak Hour Analysis

## Overview
This project analyzes online shopping order data to identify peak shopping hours during the day using Exploratory Data Analysis (EDA).

## Features
- Simulated e-commerce order data
- Extraction of hourly order patterns
- Descriptive statistics of order amounts
- Line plot visualization of peak shopping hours

## Technologies Used
- Python
- Pandas
- NumPy
- Matplotlib

## How to Run
1. Install dependencies:
   pip install -r requirements.txt
2. Run the script:
   python analysis.py

## Output
- Console statistics
- Line plot showing number of orders per hour
